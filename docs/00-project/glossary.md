# Business Glossary

| Business Term | Technical Term | Definition |
|---|---|---|
| Laba Rugi | Profit and Loss / `profit_loss_report` | Financial report showing revenue, COGS, expenses, and profit. |
| Neraca | Balance Sheet / `balance_sheet_report` | Financial position report showing assets, liabilities, and equity. |
| HPP | COGS / `cogs_amount` | Harga Pokok Penjualan. Cost of goods sold from ingredient usage or recipe standard. |
| Buku Bank | Bank Book / `bank_transactions` | Cash and bank movement ledger. |
| Penjualan Harian | Daily Sales / `daily_sales` | Daily outlet sales summary from POS export or manual input. |
| Pemakaian Bahan | Ingredient Usage / `stock_usages` | Daily recorded ingredient usage from outlet. |
| Stock Opname | Stock Count / `stock_counts` | Physical inventory count compared against system quantity. |
| Adjustment In/Out | Stock Adjustment / `stock_adjustments` | Manual stock correction with reason and approval. |
| COA | Chart of Accounts / `accounts` | Account master used by journal entries. |
| General Journal | Journal / `journals` | Accounting transaction header. |
| Journal Entry | Journal Line / `journal_entries` | Debit or credit line in a journal. |
| AP | Accounts Payable | Supplier debt. |
| AR | Accounts Receivable | Customer/outlet receivable. |
| Closing | Period Lock / `period_closings` | Process that finalizes a period and blocks changes. |
