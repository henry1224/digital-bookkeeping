# Glosarium Bisnis

| Istilah Bisnis | Istilah Teknis | Definisi |
|---|---|---|
| Laba Rugi | Profit and Loss / `profit_loss_report` | Laporan keuangan yang menampilkan revenue, HPP/COGS, expense, dan profit. |
| Neraca | Balance Sheet / `balance_sheet_report` | Laporan posisi keuangan yang menampilkan asset, liability, dan equity. |
| HPP | COGS / `cogs_amount` | Harga Pokok Penjualan. Pada MVP diposting dari actual ingredient usage; recipe dipakai sebagai pembanding standar. |
| Buku Bank | Bank Book / `bank_transactions` | Ledger pergerakan kas dan bank. |
| Penjualan Harian | Daily Sales / `daily_sales` | Ringkasan penjualan harian outlet dari export POS atau input manual. |
| Pemakaian Bahan | Ingredient Usage / `stock_usages` | Catatan pemakaian ingredient harian dari outlet. |
| Stock Opname | Stock Count / `stock_counts` | Perhitungan fisik inventory dibandingkan quantity sistem. |
| Adjustment In/Out | Stock Adjustment / `stock_adjustments` | Koreksi stock manual dengan reason dan approval. |
| COA | Chart of Accounts / `accounts` | Master akun untuk journal entries. |
| Jurnal Umum | Journal / `journals` | Header transaksi akuntansi. |
| Baris Jurnal | Journal Line / `journal_entries` | Baris debit atau credit dalam journal. |
| AP | Accounts Payable | Utang supplier/vendor. |
| AR | Accounts Receivable | Piutang customer/outlet. |
| Closing | Period Lock / `period_closings` | Proses finalisasi periode dan pemblokiran perubahan. |
| Reopen Period | Period Reopen | Pembukaan sementara closed period dengan approval Owner untuk koreksi. |
| Moving Average | Inventory Costing Method | Metode biaya rata-rata berjalan untuk valuation inventory MVP. |
| PB1 | Outlet Tax Config | Pajak/resto yang dapat dikonfigurasi per outlet jika dipakai. |
| Central Kitchen | Outlet/Storage | Pada MVP diperlakukan sebagai lokasi inventory, bukan modul produksi. |
