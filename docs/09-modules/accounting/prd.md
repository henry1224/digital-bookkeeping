# Module PRD: Accounting

## Goal

Transform operational transactions into accounting records and final financial reports.

## Features

1. General Journal
2. Journal Entry Lines
3. Buku Besar
4. Trial Balance
5. Laba Rugi
6. Neraca
7. Cash Flow Statement
8. Period Closing

## Acceptance Criteria

1. Posted journals always balance.
2. Source-generated journals link to source transaction.
3. Manual journals require approval.
4. Ledger reflects posted journals only.
5. Trial Balance total debit equals total credit.
6. Laba Rugi and Neraca can be generated per outlet and consolidated.
7. Closed period blocks edits.
